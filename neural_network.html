A neural network consists of multiple layers:

1. **Input Layer**: Receives the input data (e.g., features like `x1`, `x2`, `x3`).
2. **Hidden Layer**: Processes data through activation functions and weights, generating new features (e.g., `h1`, `h2`, `h3`).
3. **Output Layer**: Provides the final output (e.g., classification or prediction).
